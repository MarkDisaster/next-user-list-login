## .env

Create a .env file and add the variable:
```bash
NEXTAUTH_SECRET=your-secret-key
```

## Run app

Run the development server:

```bash
pnpm run dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
